# Hub Constructor — Skills comunitarios

Este módulo funciona como un marketplace de skills jurídicos creados por la comunidad. Permite descubrir, instalar y verificar skills de terceros que amplían las capacidades de Claude para Abogados. Incluye un sistema de control de calidad (QA) para garantizar la fiabilidad y seguridad de los skills antes de su uso en producción. Pensado para despachos que quieren personalizar su entorno con skills especializados y para desarrolladores que quieren contribuir al ecosistema.

## Skills disponibles

Los skills de este módulo se configuran durante la entrevista inicial y varían según las necesidades del usuario. Las funciones base del hub son:

| Función | Descripción |
|---------|-------------|
| Discovery | Busca y explora skills publicados por la comunidad según área de práctica y jurisdicción (con preferencia por skills adaptados a derecho panameño) |
| Install | Instala un skill verificado en tu entorno de trabajo local |
| QA | Ejecuta pruebas de calidad, seguridad y formato sobre un skill antes de activarlo |

## Primeros pasos

1. Ejecuta la entrevista inicial del proyecto para configurar tu perfil y preferencias.
2. Explora el catálogo de skills comunitarios disponibles para tu área de práctica.
3. Selecciona un skill y ejecuta el proceso de QA para verificar su calidad antes de instalarlo.
4. Instala el skill verificado y pruébalo en un caso de ejemplo antes de usarlo con datos reales.

## Casos de uso típicos

- Descubrimiento de un skill de análisis de cláusulas de no competencia creado por un despacho laboral.
- Instalación de un skill de cálculo de plusvalías municipales desarrollado por la comunidad fiscal.
- Ejecución de QA sobre un skill de terceros antes de incorporarlo al flujo de trabajo del despacho.
- Publicación de un skill propio para compartir con otros usuarios de la plataforma.

## Legislación de referencia

Este módulo no está vinculado a legislación específica. Los skills individuales instalados referencian su propia normativa aplicable en su documentación.

## Lo que este plugin no hace

- **No ejecuta skills no verificados automáticamente**: todo skill comunitario debe pasar el proceso de QA antes de poder ser utilizado.
- **No garantiza la exactitud jurídica de skills de terceros**: la verificación cubre seguridad, formato y estructura, pero el contenido jurídico es responsabilidad del autor.
- **No permite la ejecución de código arbitrario**: los skills operan dentro del sandbox de seguridad del sistema.
- **No aloja datos del usuario en servidores externos**: los skills se ejecutan localmente en tu entorno.
