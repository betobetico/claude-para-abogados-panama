# Consultorio Jurídico — Asistencia jurídica (Panamá)

Este módulo está diseñado para consultorios jurídicos universitarios y servicios de orientación legal gratuita en Panamá. Asiste en la toma de datos del cliente, elaboración de memorándums internos, investigación jurídica, control de plazos y redacción de cartas. Todo el trabajo producido por este módulo debe ser supervisado por un tutor académico o abogado responsable antes de ser entregado al cliente. Pensado para estudiantes en práctica forense, defensores de oficio del Órgano Judicial y ONGs con servicio de orientación legal y acceso a la justicia.

## Skills disponibles

| Skill | Comando | Descripción |
|-------|---------|-------------|
| Intake | `/intake` | Estructura la entrevista inicial con el cliente y genera la ficha del caso con los datos relevantes |
| Memorándum | `/memo` | Elabora un memorándum jurídico interno con análisis del caso, normas aplicables y opciones legales |
| Investigación | `/investigacion` | Guía la búsqueda de legislación, jurisprudencia y doctrina aplicable al caso |
| Plazos | `/plazos` | Controla los plazos relevantes del caso (prescripción, caducidad, procesales) |
| Carta | `/carta` | Redacta borradores de cartas al cliente, contraparte o administración |

## Primeros pasos

1. Ejecuta la entrevista inicial del proyecto para configurar tu perfil y preferencias.
2. Prueba `/intake` para estructurar los datos de un nuevo caso del consultorio.
3. Usa `/memo` para elaborar el análisis jurídico interno del caso.
4. Ejecuta `/plazos` para verificar que no se pase ningún plazo de prescripción o caducidad.

## Casos de uso típicos

- Entrevista inicial con un cliente que acude al consultorio por un problema de arrendamiento.
- Elaboración de un memo interno analizando las opciones legales de un caso de consumo (Ley 45 de 2007).
- Investigación de jurisprudencia de la Corte Suprema de Justicia sobre una cuestión de responsabilidad civil.
- Redacción de una carta de reclamación a un proveedor en nombre de un cliente del consultorio.

## Lo que este plugin no hace

- **No asume la defensa del cliente**: no actúa como letrado ni presenta escritos en nombre del cliente.
- **No sustituye la supervisión del tutor**: todo el trabajo debe ser revisado y aprobado por el tutor académico o abogado responsable del consultorio.
- **No emite dictámenes vinculantes**: los análisis son orientativos y de carácter formativo.
- **No accede a bases de datos de pago**: no consulta directamente el Registro Judicial del Órgano Judicial ni bases de datos jurídicas comerciales.
