# Privacidad — Protección de datos (compliance)

Este módulo se centra en el cumplimiento normativo en materia de privacidad y protección de datos personales en Panamá. Asiste en la gestión de derechos de los titulares, contratos de encargado de tratamiento, evaluaciones de impacto y análisis de brechas de cumplimiento frente a la Ley 81 de 2019 y su Decreto Ejecutivo reglamentario 285 de 2021. Orientado a responsables de cumplimiento, encargados de protección de datos y abogados especializados en privacidad que necesitan estructurar su programa de compliance.

## Skills disponibles

| Skill | Comando | Descripción |
|-------|---------|-------------|
| Derechos | `/derechos` | Gestiona solicitudes de derechos del titular (acceso, rectificación, cancelación/supresión, oposición, portabilidad) |
| Encargo de tratamiento | `/encargo` | Revisa o genera contratos de encargado de tratamiento conforme a la Ley 81 de 2019 |
| EIPD | `/eipd` | Guía la elaboración de una Evaluación de Impacto en Protección de Datos |
| Triaje | `/triaje` | Clasifica un tratamiento de datos según nivel de riesgo y obligaciones aplicables |
| Gap analysis | `/gap` | Identifica brechas de cumplimiento respecto a la Ley 81 de 2019 y el Decreto Ejecutivo 285 de 2021 |

## Primeros pasos

1. Ejecuta la entrevista inicial del proyecto para configurar tu perfil y preferencias.
2. Prueba `/triaje` con la descripción de un tratamiento de datos para clasificar su nivel de riesgo.
3. Usa `/gap` para auditar el estado de cumplimiento de una organización.
4. Ejecuta `/derechos` cuando recibas una solicitud de un titular para gestionar los plazos y la respuesta.

## Casos de uso típicos

- Clasificación de riesgo de un nuevo tratamiento de datos de empleados.
- Elaboración de una evaluación de impacto para un sistema de videovigilancia o de monitorización de productividad.
- Revisión de un contrato de encargado del tratamiento de un proveedor cloud.
- Auditoría de cumplimiento de protección de datos de una organización antes de una actuación de la ANTAI.

## Legislación de referencia

- **Ley 81 de 2019** — Sobre Protección de Datos Personales (publicada en la Gaceta Oficial de Panamá)
- **Decreto Ejecutivo 285 de 2021** — que reglamenta la Ley 81 de 2019

## Lo que este plugin no hace

- **No realiza auditoría técnica de seguridad**: no ejecuta pentesting, escaneos de vulnerabilidades ni revisiones de infraestructura TI.
- **No certifica esquemas de seguridad**: no emite certificaciones de seguridad de la información.
- **No notifica brechas a la ANTAI**: no envía comunicaciones formales a la Autoridad Nacional de Transparencia y Acceso a la Información.
- **No sustituye el criterio profesional** del encargado de protección de datos o abogado: las recomendaciones deben ser validadas por un profesional cualificado.
